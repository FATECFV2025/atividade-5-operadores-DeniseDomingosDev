Projeto Calculadora em Java

Objetivo:
Implementar uma classe que simule operações de uma calculadora com operadores:
- Aritméticos
- De Atribuição
- Lógicos
- De Comparação

Estrutura do Projeto:
- **Operadores.java** → Contém todos os métodos de cálculo e comparação.  
- **Calculadora.java** → Executa os testes e interações com o usuário.
